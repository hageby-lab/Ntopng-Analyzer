# Руководство по внесению вклада

## Стиль кода
- Используйте форматирование Black
- Следуйте guidelines PEP 8
- Добавляйте type hints для новых функций
- Пишите docstrings для всех функций

## Процесс Pull Request
1. Форкните репозиторий
2. Создайте ветку функции (`git checkout -b feature/AmazingFeature`)
3. Закоммитьте изменения (`git commit -m 'Add AmazingFeature'`)
4. Запушьте ветку (`git push origin feature/AmazingFeature`)
5. Откройте Pull Request

## Тестирование
- Добавляйте тесты для новой функциональности
- Убедитесь, что все тесты проходят
- Обновляйте документацию по необходимости
